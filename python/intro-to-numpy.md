NumPy is a library for numerical computing in Python.

### Installing NumPy

Install using pip:

```plaintext
pip install numpy
```

### Creating and Manipulating Arrays

Example:

```python
import numpy as np

array = np.array([1, 2, 3, 4])
print("Array:", array)
print("Shape:", array.shape)
print("Sum:", array.sum())
```

### Expected Output

```plaintext
Array: [1 2 3 4]
Shape: (4,)
Sum: 10
```

NumPy is essential for scientific computing and data analysis.
